Machine Learning

Previsão de Vendas com Machine Learning

Neste projeto, a metodologia CRISP-DM foi aplicada para desenvolver uma solução de machine learning customizada para prever vendas nas lojas Rossmann. O objetivo principal foi aprimorar e agilizar os processos de tomada de decisão dos executivos da empresa.

O projeto percorreu todas as fases essenciais de um projeto de machine learning, como: descrição de dados, análise exploratória, feature engineering, feature selection, seleção do modelo, ajuste de hiperparâmetros, levando finalmente à implantação do modelo XGBoost em um ambiente de produção.

O resultado deste projeto é um modelo XGBoost, acessível por meio de uma API, integrado a um bot do Telegram para facilitar a comunicação eficiente e a agilidade operacional.
